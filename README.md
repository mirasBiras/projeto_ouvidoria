# Sistema de Ouvidoria

Aplicação desenvolvida em Python com integração ao MySQL para gerenciamento de reclamações.

## Funcionalidades
- Cadastro de reclamações
- Listagem completa
- Busca por ID
- Atualização de registros
- Remoção
- Contagem total

## Tecnologias utilizadas
- Python
- MySQL
- Git e GitHub
